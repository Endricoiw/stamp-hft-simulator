# 📮 stamp-hft-simulator
A high-frequency trading simulator for rare Victorian postage stamps featuring a sentiment engine and rumor-based volatility.

## 🧐 What is this?
This script simulates a micro-market for legendary stamps like the **Blue Mauritius** and the **British Guiana 1c Magenta**. Prices aren't random; they are driven by a **Sentiment Engine** that mimics collector hype and paper stability.

## 🚀 Key Features
*   **Live Price Ticker**: Real-time updates with green/red deltas.
*   **Rumor Engine**: Random events like "Attic Finds" (supply shock) or "Forgery Scandals" (market crash).
*   **Lab Analysis**: Simulates chemical pH testing and perforation scoring for physical asset quality.
*   **Brownian Motion logic**: Price fluctuations based on sine-wave sentiment and random noise.

## 🛠️ Installation & Usage
1. Ensure you have **Python 3.x** installed.
2. Clone this repo or download `main.py`.
3. Run the simulator:
   ```bash
   python main.py
   ```
4. Press `Ctrl + C` to close the market (if you can handle the losses).

## ⚠️ Disclaimer
This is a **simulation**. Do not attempt to use this as financial advice for real-world stamp collecting. The "Blue Mauritius" prices in this script are for entertainment purposes only. Keep your physical stamps in a cool, dry place.
